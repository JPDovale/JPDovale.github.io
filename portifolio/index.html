Baixando jQuery Cópias compactadas e descompactadas de arquivos jQuery estão disponíveis. O arquivo descompactado é melhor usado durante o desenvolvimento ou depuração; o arquivo compactado economiza largura de banda e melhora o desempenho na produção. Você também pode baixar um arquivo de mapa de origem para uso ao depurar com um arquivo compactado. O arquivo de mapa não é necessário para que os usuários executem o jQuery, ele apenas melhora a experiência do depurador do desenvolvedor. A partir do jQuery 1.11.0/2.1.0 o //# sourceMappingURLcomentário não é incluído no arquivo compactado. Para baixar localmente esses arquivos, clique com o botão direito do mouse no link e selecione "Salvar como..." no menu.
